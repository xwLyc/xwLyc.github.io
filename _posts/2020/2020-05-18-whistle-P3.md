#### 调试 whistle 移动端vconsole调试

这个就有点牛逼了，可以给任何项目添加 `vconsole` 展示，免去了代码安装。看图中耀眼的小绿条...


![pic](../../../images/2020/05/21.png)



直接安装相关插件即可

    sudo npm install whistle.inspect -g

然后回到whistle的规则配置里

![pic](../../../images/2020/05/22.png)


这时候再去移动端访问相关页面，就发现已经有了，是不是巨方便！无需项目代码安装！
